## 介绍

本仓库提供了一份重要的资源文件，即《充电桩标准(2016国标)》，涵盖了交流和直流充电桩的系列标准。这些标准是充电桩行业的重要参考，确保了充电桩的设计、制造和使用符合国家规范，保障了充电桩的安全性和可靠性。

## 文件内容

该资源文件详细描述了以下内容：

- **交流充电桩标准**：包括交流充电桩的技术要求、测试方法、安全规范等。
- **直流充电桩标准**：涵盖直流充电桩的技术要求、测试方法、安全规范等。
